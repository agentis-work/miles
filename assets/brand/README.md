# Miles Brand Assets

This folder contains the source and exported brand assets for **Miles - Your AI travel guide**.

## Source SVG masters

- `svg/miles-full.svg`
  - Full lockup (wordmark + silhouette + abstract road)
  - Use on large surfaces: splash, hero overlays, web header (desktop)
- `svg/miles-mark.svg`
  - Simplified "M" mark
  - Use on compact surfaces: app icon, small headers, tight UI spaces
- `svg/miles-favicon.svg`
  - Tiny-size optimized mark for favicon exports

All SVG masters use `currentColor` so they can be themed.
